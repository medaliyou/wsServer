## wsServer
 C++ Websocket Server based on QtWebsocket API.
### Building steps: 
#### create a building directory for example 'build'
##### `$ mkdir build`
#### CD to the build output directory (KEY STEP)
##### `$ cd build`
#### Call qmake referencing the .pro file
##### `$ qmake ../wsServer.pro`
#### Call make 
##### `$ make`
